## How to use as a typical user
- This Project is a backend project that allows you to to track customers, service tickets, and mechanics. 
- To test out the backend you can use postman to access the customers, tickets, and mechanics. 
- Before you open postman to get the site up and running you can open a new terminal and type in python app.py and that will start the backend. 
- once the backend is going go to postman and you can create customers, tickets, and mechanics by creating a a new request and selecting post and adding your data. 
- You can also get specific customers, tickets, and mechanics by changing the path and adding the id number of the item you wish to get access too. 
- You can also get all of the items. 
- If you'd like to edit any of the items you can do a put request and this will allow you to change any information. 
- Finally for customers and Mechanics you can use a delete request to get rid of a specific customer or mechanic.
